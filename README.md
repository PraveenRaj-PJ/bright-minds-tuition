**Bright Minds Tuition Center Portfolio**

A modern, responsive portfolio website for Bright Minds Tuition Center, designed to showcase courses, services, and educational values.

**About the Project**

This project was initially generated using Claude AI and further customized and enhanced with my own development knowledge to improve design, usability, and overall user experience.

**Features**

Responsive and user-friendly design
Clean and structured layout
Course and service showcase
Contact form integrated with Google Sheets for data collection

**Technologies Used**

HTML
CSS
JavaScript

**Google Sheets (for form data storage)**

🔗 Google Sheets Integration

Form submissions are automatically stored in Google Sheets using a connected script, making it easy to manage and track responses.
